# Project Summary
El proyecto **Asesor de Pasajes Sublo** es una aplicación web diseñada para facilitar la consulta y gestión de pasajes para empleados de Aerolíneas Argentinas. Proporciona funcionalidades como autenticación de usuarios, gestión de beneficiarios, consulta de pasajes disponibles y normativa relacionada, todo dentro de una interfaz profesional y corporativa que mejora la experiencia del usuario. Recientemente se ha implementado un sistema de registro de usuarios que permite a nuevos empleados crear cuentas asociadas a su legajo.

# Project Module Description
- **Sistema de Autenticación**: Permite el inicio de sesión y la gestión de sesiones con localStorage, ahora incluye registro de usuarios.
- **Dashboard Principal**: Muestra un resumen del estado de beneficios, antigüedad y alertas relevantes.
- **Gestión de Beneficiarios**: Funcionalidades para agregar, editar y eliminar beneficiarios, incluyendo validaciones y clasificaciones.
- **Consulta de Pasajes Disponibles**: Permite verificar la elegibilidad de pasajes y listar opciones disponibles con detalles.
- **Normativa Completa**: Proporciona información detallada sobre grupos familiares, tipos de pasajes y restricciones.
- **Registro de Usuarios**: Nueva página que permite a los empleados registrarse, incluyendo validaciones de datos personales y laborales.

# File Description Inventory
- **README.md**: Documentación del proyecto.
- **index.html**: Estructura HTML principal de la aplicación.
- **App.tsx**: Componente principal que define las rutas y la estructura de la aplicación.
- **package.json**: Contiene las dependencias y scripts necesarios para el proyecto.
- **tailwind.config.ts**: Configuración para el uso de Tailwind CSS.
- **vite.config.ts**: Configuración para el bundler Vite.
- **AuthContext.tsx**: Contexto para la gestión de autenticación de usuarios.
- **Login.tsx**: Página de inicio de sesión para usuarios existentes.
- **Register.tsx**: Nueva página para el registro de nuevos usuarios.

# Technology Stack
- **React**: Biblioteca para construir interfaces de usuario.
- **TypeScript**: Superset de JavaScript que añade tipado estático.
- **Shadcn-ui**: Biblioteca de componentes UI.
- **React Router**: Manejo de enrutamiento en la aplicación.
- **LocalStorage**: Persistencia de datos en el navegador.
- **Tailwind CSS**: Framework de CSS para estilos.

# Usage
1. Instalar dependencias:
   ```bash
   cd /workspace/shadcn-ui && pnpm i
   ```
2. Ejecutar linter:
   ```bash
   cd /workspace/shadcn-ui && pnpm run lint
   ```
3. Construir el proyecto:
   ```bash
   cd /workspace/shadcn-ui && pnpm run build
   ```
